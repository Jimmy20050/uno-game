# Uno Game - Play Online

A fully functional Uno card game with AI opponent, built with HTML, CSS, and JavaScript.

## 🎮 Features

- Complete Uno rules (number cards, skip, reverse, draw 2, wild, wild draw 4)
- Smart AI opponent with strategic gameplay
- Beautiful modern UI with Uno-themed background
- Responsive design for all screen sizes
- UNO button for calling when you have one card
- Win/lose detection with celebration screen

## 🚀 How to Deploy Online

### Option 1: Netlify Drop (Easiest - Recommended)
1. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop the entire `uno-game` folder onto the page
3. Wait a few seconds for deployment
4. You'll get a free URL like `your-game-name.netlify.app`
5. Share that URL with anyone!

### Option 2: GitHub Pages
1. Create a GitHub account if you don't have one
2. Create a new repository
3. Upload all files from the `uno-game` folder
4. Go to Settings > Pages
5. Select "main" branch and save
6. Your site will be at `your-username.github.io/repository-name`

### Option 3: Vercel
1. Go to [https://vercel.com](https://vercel.com)
2. Sign up for a free account
3. Click "New Project"
4. Upload the `uno-game` folder
5. Deploy and get your URL

### Option 4: Surge.sh (Command Line)
```bash
npm install -g surge
cd uno-game
surge
```

## 🎯 How to Play

1. Click "Play vs AI" to start
2. Click on cards in your hand to play them
3. Match cards by color, number, or symbol
4. Click the draw deck if you can't play
5. Click "UNO!" when you have one card left
6. First to empty their hand wins!

## 📁 Files

- `index.html` - Main game structure
- `styles.css` - Styling and animations
- `game.js` - Game logic and AI

## 🎨 Customization

You can easily customize:
- Colors in `styles.css`
- AI difficulty in `game.js`
